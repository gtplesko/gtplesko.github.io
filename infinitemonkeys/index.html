Infinite Monkeys

A Genetic Algorithm on text

For this project, I will allow you to tinker with the values to see how the algorithm works best. I have preset the conditions for what I used in testing. I haven't set limits to values.

Target - What your goal string is I do not suggest exceeding 140 characters.

Pool Size - The amount of children that are in each generation Value 20-200, this will eat your processor

Generations - How many times the algorithm compounds Test it first with a low number

Mutation Order - How much text is preserved when a mutation happens (Higher value, less mutation) Suggested Value 1-5

This area will be updated with every 20th generation.
